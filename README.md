**Script Won't Run:**
```bash
# Make sure it's executable

```

**Permission Denied:**
```bash
# Run with proper permissions
sudo ./ultrabunt.sh
```

**Package Installation Fails:**
```bash
# Update package lists first
sudo apt update

# Fix broken dependencies
sudo apt --fix-broken install
```
